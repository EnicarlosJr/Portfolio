# Portfolio - Enicarlos Junior

Portfolio profissional em React, Vite e Tailwind, focado em recrutamento para vagas junior de desenvolvimento web.

## Stack

- React + Vite
- Tailwind CSS
- Framer Motion
- Lucide Icons
- Funcao serverless em `api/contact.js`

## Funcionalidades implementadas

- Timeline visual completa em `/experience`
- Mapa tecnico interativo na Home
- Terminal flutuante global acessivel em qualquer pagina
- Formulario com chamada para funcao serverless e fallback por e-mail
- SEO dinamico por rota, JSON-LD, `sitemap.xml` e `robots.txt`
- Experiencia DECOM adicionada ao portfolio
- Painel antigo removido da navegacao e do fluxo principal

## Rodar localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
